P: Que herramientas son necesarias para realizar un Envío?

Respuesta

Para poder enviar una tener disponible la herramienta CURL y utilizar tu consola de comandos. También puedes usar otras herramientas como Postman u otras alternativas que soporten el envío de peticiones web con ficheros.

P: Donde puedo ver el link para realizar un Envío?

Respuesta

En la descripción de una Tarea encontrarás el botón Realizar Envío y al hacer click se abrirá un popup con el código a copiar para hacer una petición web via curl.

P: La ejecución es correcta pero la marca como inválida

Respuesta

Verifica la descripción de la Tarea ya que el formato de la solución debe ser el que espera Pizarra. Además, puede que tu solución no contemple algún caso específico del problema.

Si después de varios intentos crees que el problema es otro ponte en contacto con el Administrador de Pizarra.

P: Como me puedo unir a un equipo?

Respuesta

Pide a un compañero de clase que entre a su perfil de usuario, busque la URL única y te la comparta. Una vez que navegues a esa URL con tu navegador ya formarás parte del equipo!
